Calls the listeners having the specified key. If one of the listeners will return False it will stop all the listeners executions for the provided key.

Decorator for configuration setup functions. For the configuration type the function will be searched for the return annotation and consider that as the type, if no annotation is present than this setup function is not known by return type. For impl classes create the setup functions for the associated API classes. The name of the setup functions that will be generated are formed based on the provided formatter. To create a setup function a class from the impl classes has to inherit the api class.
