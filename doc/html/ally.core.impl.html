impl Package¶

Created on Jun 18, 2011

@package: ally core @copyright: 2011 Sourcefabric o.p.s. @license: http://www.gnu.org/licenses/gpl-3.0.txt @author: Gabriel Nistor

Has modules that implement specifications for the server. This implementations range from organizational classes like the assmebler, node and invoker to actual content handling classes like the encoders and decoders. This implementations are general by nature they are not binded to the HTTP protocol bu some are binded to text representations for instance the XML encoder.

class ally.core.impl.assembler.AssembleDelete[source]¶

Bases: ally.core.impl.assembler.AssembleOneByOne

Resolving the DELETE method invokers. Method signature needs to be flagged with DELETE and look like: boolean % ([...AnyEntity.Property], [AnyEntity.Id]) The last property needs to target the actual entity that is being deleted. !!!Attention the order of the mandatory arguments is crucial since based on that the call is placed in the REST Node tree.
