ally.design.context.optional(doc, *types)[source]¶

Construct an optional attribute for the context. The optional attribute means that the context is valid even if there is no value for the attribute.

class ally.design.processor.Chain(calls)[source]¶

Bases: builtins.object

A chain that contains a list of processors (callables) that are executed one by one. Each processor will have
the duty to proceed with the processing if is the case by calling the chain.
isConsumed()[source]¶

Checks if the chain is consumed.

@return: boolean
True if all processors from the chain have been executed, False if a processor from the chain has stopped the execution of the other processors.
proceed()[source]¶

Indicates to the chain that it should proceed with the chain execution after a processor has returned. The proceed is available only when the chain is in execution. The execution is continued with the same arguments.

process(**keyargs)[source]¶

Called in order to execute the next processors in the chain. This method will stop processing when all the processors have been executed.

@param keyargs: key arguments
The key arguments that are passed on to the next processors.
