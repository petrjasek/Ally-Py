Detect if part or all of the given path points to a ZIP file

@param filePath: string
The full path to the resource
@return: tuple(string, string)
Returns a tuple with the following content: 1. path to the ZIP file in OS format (using OS path separator) 2. ZIP internal path to the requested file in ZIP format
