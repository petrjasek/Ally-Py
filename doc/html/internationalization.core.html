getGlobalAsDict(locale)[source]¶

Provides the messages for the whole application and the given locale in the form of a dictionary. The dictionary has the following structure: {

“mydomain” : {

// po header fields “” : {

“plural-forms” : ”...”, “lang” : “en”,

}, // all the msgid strings and translations “context:msgid” : [ “msgid_plural”, “translation”, “plural_translation” ],

},

}

@param locale: string
The locale for which to return the translation.
@return: dict
The dictionary containing the translation.
